# gRPC-Client-Server
#### gRPC client server using NodeJS and MYSQL

#

>**To run this project:**

Change directory to ***server***
```
cd server
```
Install the dependencies
```
yarn install
```
Start MYSQL database
```
sudo /opt/lampp/xampp start 
```
Start server
```
yarn start
```
